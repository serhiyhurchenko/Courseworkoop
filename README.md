Курсова робота
Робота з принципами ООП в С#
Мета:
Закріпити навички роботи з принципами ООП на мові програмування С#, використання інтерфейсів, наслідування, поліморфізму, інкапсуляції та правильного розподілення відповідальність між класами.
Завдання:
Реалізувати систему для "Хрестики-нулики" 
Система має бути реалізована за принципами ООП. 
Система має включати в себе:
-	зберігання даних про користувача в імітацію бази даних;
-	використання принципів ООП;
-	програмний продукт має бути розділений на рівні: комунікація з користувачем, основна логіка програми, логіка роботи з даними(БД).

Для "Хрестики-нулики" треба додатково реалізувати:
-	логіку гри;
-	реєстрація користувача;
-	вхід до системи, якщо користувач вже зареєстрований;
-	перевірка рейтингу;
-	можливість зіграти (в результаті гри має оновитись рейтинг);
-	перегляд історії ігор;

Вимоги:
•	Правильно розроблено класи та інтерфейси для комунікації з користувачем, логіки програми та роботи з даними.
•	Правильно реалізовано наслідування, інкапсуляція та поліморфізм. 
•	Розбиття коду на рівні. 
•	Правильно розподіллено обов’язки класів. 
•	Правильно реалізована взаємодія між класами.
•	Демонстрація створенні класів, демонстрація роботи їх методів.
•	Записка.

Критерії оцінювання:
•	Правильно написані класі, їх властивості та методи.
•	Правильність взаємодії між класами.
•	Розуміння концепту інтерфейсів, поліморфізму, інкапсуляції, наслідування та розподілення відповідальності між класами.
•	Правильність маніпулювання даними.
